# Hogan's Portfolio

Personal portfolio built with Next.js, React, Tailwind CSS, and Framer Motion. Deployed on Vercel.

## Tech Stack

- Next.js 15 (App Router)
- React 19
- Tailwind CSS 4
- Framer Motion
- Three.js / React Three Fiber

## Getting Started

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

## Scripts

```bash
npm run dev    # Start dev server
npm run build  # Production build
npm run start  # Run production server
npm run lint   # Lint
```

## Project Structure

- `src/app` - App Router pages and layout
- `src/components` - UI, layout, and section components
- `public` - Static assets

## Updating Content

- Project cards: `src/components/sections/Projects.tsx`
- Social links (GitHub/LinkedIn): `src/components/layout/Footer.tsx`
